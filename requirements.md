## Installing Python3 on EC2

* Install Python3:

      sudo yum list | grep python3
      sudo yum install python36
* Install influxdb
  
      https://www.easyitblog.info/2017/11/14/influxdb-in-iot-world-aws-part-2/
* Install git

      sudo yum install git

* Install Flask

      sudo python3 -m pip install flask
